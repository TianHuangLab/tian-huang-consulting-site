# Tian HUANG — Tracking & Attribution Consultant

Personal consulting website for a French-market Tracking & Attribution Consultant.

The site focuses on:

- GA4, GTM and Consent Mode audits
- Tracking issue diagnosis and correction
- Revenue and attribution reconciliation
- Dashboard data quality
- Measurable marketing websites for small businesses

## Tech Stack

- Astro
- Tailwind CSS
- TypeScript
- Netlify
- Cal.com embed for booking

## Local Development

```bash
npm install
npm run dev
```

The local site usually runs at:

```text
http://localhost:4321
```

## Checks

```bash
npm run check
npm run build
```

`npm run check` runs Astro diagnostics, ESLint and Prettier checks.

## Deployment

The site is deployed on Netlify.

Build settings:

```text
Build command: npm run build
Publish directory: dist
Node version: 22
```

Production URL:

```text
https://huang-tian.netlify.app
```
